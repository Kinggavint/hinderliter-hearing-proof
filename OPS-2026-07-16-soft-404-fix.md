# Ops record — 2026-07-16 soft-404 fix

## What broke
The WordPress → static-site cutover (2026-01-25 through 2026-04-22, pre-Apex) left 134 previously-indexed WP URLs returning HTTP 200 with the homepage body instead of proper 301 redirects or 404s.

## Why it broke
- CloudFront distribution `E351Q8RD58U3KZ` had `CustomErrorResponses: [{ ErrorCode: 404, ResponsePagePath: /index.html, ResponseCode: 200 }]`. This is SPA-fallback semantics on a non-SPA site.
- S3 bucket `hinderliter-hearing-site` had `ErrorDocument: index.html` in its website config, which doubles down on the same problem.

## What was shipped
1. CloudFront `CustomErrorResponses[0]` set to `{ ErrorCode: 404, ResponsePagePath: /404.html, ResponseCode: 404, ErrorCachingMinTTL: 300 }`.
2. S3 website config `ErrorDocument` changed from `index.html` to `404.html`.
3. New `/404.html` uploaded (2.8 KB, noindex, patient-facing recovery grid).
4. 270 S3 objects uploaded with `WebsiteRedirectLocation` metadata, covering all 135 orphaned URLs from the archived WP sitemaps in both trailing-slash and no-slash variants.
5. `/*` invalidation on `E351Q8RD58U3KZ`. Verified live via curl.

## Do not undo
- The old CloudFront error-response setting (`404 → /index.html [200]`) was actively harmful. Do not restore it in any deploy script or "restore defaults" action.
- The S3 website `ErrorDocument = index.html` was actively harmful. Do not restore it.
- The 270 redirect objects are empty content-length-0 objects with metadata; they must not be overwritten by `aws s3 sync` runs from the repo (they have no corresponding local file). If a future sync uses `--delete` it will remove them. **Do not add `--delete` to the sync script.**

## Expected impact
- Google will recrawl the 301'd URLs over 30-90 days and consolidate authority into `/hearing-aids.html`, `/hearing-test-near-me.html`, `/tinnitus-evaluation-management.html`, and `/services.html`.
- Onspire's traffic-drop narrative should not be pinned on the Apex redesign; the pre-Apex cutover is what caused it. Apex fixed the actual root cause.
